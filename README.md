# Log Analysis Tool

A Java-based log analysis project that simulates real-world log processing by classifying events such as ERROR, WARNING, and INFO.

## Features
- Processes and classifies 1000+ simulated log entries
- Supports keyword-based log filtering
- Organizes events by category for faster review
- Built using object-oriented design principles

## Technologies Used
- Java

## Purpose
This project was built to model how log data can be processed and categorized to improve debugging and system monitoring workflows.

## Files
- `Main.java`
- `LogAnalyzer.java`

## How to Run
1. Open the project in VS Code or any Java IDE
2. Compile the Java files
3. Run `Main.java`
